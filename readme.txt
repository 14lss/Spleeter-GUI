Requires Spleeter: https://github.com/deezer/spleeter
Requires python: https://www.python.org/downloads/

Download latest release and extract. 
Run Spleeter.exe and select an mp3 file to separate. 
Stems will be created in /spleeter.exe directory/audio_output

Make sure spleeter_logo.ico is installed in the same directory as spleeter.exe
